Capítulo 1 Introducción

Este bookdown sirven como notas para el curso Modelización estadística avanzada con R: análisis de datos longitudinales impartido en el Insituto Aragonés de Ciencias de la Salud

El contenido del curso tiene los siguientes temas:

  • Módulo 1: Modelos lineales para datos longitudinales continuos (I)
    • Visualización de datos longitudinales
    • ANOVA para medidas repetidas
  • Módulo 2: Modelos lineales para datos longitudinales continuos (II)
    • MANOVA
    • Modelos mixtos
  • Módulo 3: Análisis de supervivencia con eventos recurrentes (I)
    • Breve introducción al análisis de supervivencia
    • Modelos condicionales y marginales
    • Modelos de fragilidad
  • Módulo 4: Modelización conjunta de datos longitudinales y de supervivencia
    • Modelos de fragilidad
    • Modelo general para eventos recurrentes
